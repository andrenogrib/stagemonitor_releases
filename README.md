# Stage Monitor — Releases

Public download host for **Stage Monitor**, a cross-platform (Windows + macOS)
ProPresenter-style stage display with a GrandMA2-style cue list, multi-screen + NDI
output, MIDI/OSC/Art-Net/sACN/SMPTE timecode and a LAN web-viewer.

➡️ **Download the latest version from the [Releases](../../releases/latest) page.**

- **Windows** — `Stage Monitor-Setup-<version>.exe` (NSIS installer)
- **macOS (Apple Silicon)** — `Stage Monitor-<version>-arm64.dmg`

The application source is maintained privately; this repository only hosts the
installable builds. Website & guide: see the project site.
